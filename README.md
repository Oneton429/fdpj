# 自动评教脚本

## 功能

自动填写评教问卷

注意：暂不支持需要输入文字的必填项

## 使用方法

进入评教平台（ce.fudan.edu.cn）—— 我的评价任务 —— 未完成页面，`F12`打开调试工具，将本仓库`main.js`中的代码复制进控制台（Console）回车运行即可。
